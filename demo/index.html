API Properties && Methods

Name Description
Properties
$.drop.nS namespace of plugin
$.drop.version version of plugin
$.drop.dP Object default parametrs
$.drop.drp system object of plugin
Methods
open $.drop([various type], [options])

First parameter can be in various types, examples:
$.drop([
    {href : '1.jpg', header : 'Title first image'},
    {href : '2.jpg', header : 'Title second image'}
]); - array containing objects
$.drop( {href : '1.jpg', title : 'Lorem lipsum'} ); - single object
$.drop( ['1.jpg', '2.jpg'] ); - array containing links as strings
$.drop( '1.jpg' ); - string as source link
$.drop( [jQuery object] );
$.drop( '<h1>Lorem lipsum</h1>' );

close $.drop.close([selector|jQuery object])

if call without parameters closing all opened popups
cancel $.drop.cancel()

Cancel loading image or abort ajax request
update $.drop.update([selector|jQuery object])

if call without parameters updated (reposition, resize) all opened popups
setMethod $.drop.setMethod(name, value)

extend default methods
getMethods $.drop.getMethods()

return only public methods (without prefix '_')
setParameters $.drop.setParameters()

extend Default parameters
setThemes $.drop.setThemes({name: css(string)}[, {}, ...])
showLoading $.drop.showLoading()

Shows loading
hideLoading $.drop.hideLoading()

Hides loading
setUrlMethods $.drop.setUrlMethods(url to files of methods)
require $.drop.require('filename'[, ...])

next methods available only if exists gallery method
next $.drop.next([rel])

Navigate to next gallery item (for visible popup galleries), or only on gallery by "rel"
prev $.drop.prev([rel])

Navigate to prev gallery item (for visible popup galleries), or only on gallery by "rel"
jumpto $.drop.jumpto(index[, rel])

Navigate to gallery item by index (for visible popup galleries), or only on gallery by "rel". Begining at 0.
play $.drop.play([rel])

Start or stop (if already running) slideshow (for visible popup galleries), or only on gallery by "rel"
