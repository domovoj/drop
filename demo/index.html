Options

For set any option can use three methods priority these methods of set parameter(-s) sort in order:

  1. By set default parametrs $.drop.setParameters({object of options})

    but this set option(-s) need make before initialization detail about initialization

  2. By initializtion $(selector initialization).drop({object of options})

  3. By data atribute with html or code, for example:
    html (data-drop):
    <a href="#" id="pop-up-refer" data-drop="#inline">inline</a>
    code:
    $('#pop-up-refer').data('drop', '#inline');

    but set this option(-s) need make before initialization detail about initialization

For auto initialization need that element had html atributes: data-drop or data-html (they may be without values)

API Properties && Methods

Name Description
Properties

Methods
open

$.drop([various type], [options])

First parameter can be in various types, examples:
$.drop([
    {href : 'with_pattern.php', header : 'Title first document'},
    {href : 'notify.php', header : 'Title second document'}
]); - array containing objects
$.drop( {href : 'with_pattern.php', header : 'Lorem lipsum'} ); - single object
$.drop( ['1.jpg', '2.jpg'] ); - array containing links as strings (only images)
$.drop( '1.jpg' ); - string as source link (only images)
$.drop( [jQuery object] );
$.drop( '<h1>Lorem lipsum</h1>' );

close

$.drop.close([selector|jQuery object], [force])

if call without parameters closing all opened pop-ups, if force: true pop-ups closed without effect
cancel

$.drop.cancel()

Cancel loading image or abort ajax request
update

$.drop.update([selector|jQuery object])

if call without parameters updated (reposition, resize) all opened pop-ups
setMethod

$.drop.setMethod(name, context of function)

extend default methods

$.drop.getMethods()

return only public methods (without prefix '_')

loads js - files

setUrlMethods

$.drop.setUrlMethods('url to files of methods')

next methods available only if exists gallery method
next

$.drop.next([rel])

Navigate to next gallery item (for visible pop-up galleries), or only one gallery identify by "rel"
prev

$.drop.prev([rel])

Navigate to prev gallery item (for visible pop-up galleries), or only one gallery identify by "rel"
jumpto

$.drop.jumpto(index[, rel])

Navigate to gallery item by index (for visible pop-up galleries), or only one gallery identify by "rel". Begining at 0.
play

$.drop.play([rel])

Start or stop (if already running) slideshow (for visible pop-up galleries), or only one gallery identify by "rel"
