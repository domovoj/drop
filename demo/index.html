Options

For set any option can use three methods priority these methods of set parameter(-s) sort in order:

  1. By set default parametrs $.drop.setParameters({object of options})

    but this set option(-s) need make before initialization detail about initialization

  2. By initializtion $(selector initialization).drop({object of options})

  3. By data atribute with html or code, for example:
    html (data-drop):
    <a href="#" id="popup-refer" data-drop="#inline">inline</a>
    code:
    $('#popup-refer').data('drop', '#inline');

    but set this option(-s) need make before initialization detail about initialization

For auto initialization need that element had html atributes: data-drop or data-html (they may be without values)

Methods

init

$('selector of element which will open popup').drop(['init' | object of parametr])

Initialize plugin on element which will opens popup
destroy

$('selector of element which opens popup').drop('destroy')

Off plugin on element which opens popup
open

$('selector of element which opens popup').drop('open')

Opens popup
close

$('selector of popup').drop('close')

Closes popup
update

$('selector of popup').drop('update')

Reposition and resize popup

API Properties && Methods

Properties
$.drop.nS namespace of plugin
$.drop.version version of plugin
$.drop.dP Object default parametrs
$.drop.drp system object of plugin
Methods
open

$.drop([various type], [options])

First parameter can be in various types, examples:
$.drop([
    {href : 'with_pattern.php', header : 'Title first document'},
    {href : 'notify.php', header : 'Title second document'}
]); - array containing objects
$.drop( {href : 'with_pattern.php', header : 'Lorem lipsum'} ); - single object
$.drop( ['1.jpg', '2.jpg'] ); - array containing links as strings (only images)
$.drop( '1.jpg' ); - string as source link (only images)
$.drop( [jQuery object] );
$.drop( '<h1>Lorem lipsum</h1>' );

close

$.drop.close([selector|jQuery object], [force])

if to call without parameters closing all opened popups, if force: true popups closed without effect
cancel

$.drop.cancel()

Cancels load for image or aborts ajax request
update

$.drop.update([selector|jQuery object])

if to call without parameters updated (reposition, resize) all opened popups
setMethod

$.drop.setMethod(name, context of function)

extends default methods
getMethods

$.drop.getMethods()

returns only public methods (without prefix '_')
setParameters

$.drop.setParameters(object)

extends Default parameters
setThemes

$.drop.setThemes({name: string, text: 'css(string)', file: 'full name of file'}[, {}, ...])

extends Default themes
showLoading

$.drop.showLoading()

Shows loading
hideLoading

$.drop.hideLoading()

Hides loading
require

$.drop.require(['filename', ...], callback)

loads js - files

next methods available only if exists gallery method
next

$.drop.next([rel])

Navigates to next gallery item (for visible popup galleries), or only one gallery identify by "rel"
prev

$.drop.prev([rel])

Navigates to prev gallery item (for visible popup galleries), or only one gallery identify by "rel"
jumpto

$.drop.jumpto(index[, rel])

Navigates to gallery item by index (for visible popup galleries), or only one gallery identify by "rel". Begining at 0.
play

$.drop.play([rel])

Starts or stops (if already running) slideshow (for visible popup galleries), or only one gallery identify by "rel"

Events

dropBefore el.on('dropBefore', function(event, object){})

Where el can be $(selector of popup) and/or $(document) (if you haven't selector of popup)

The event happens before opening popup. The event sent to popup which opens popup (if popup not exists in moment sample handler that necessary sample handler to $(document)).
Second argument in handler contains object.

  • options

    Object of options the popup

  • drop

    jQuery element (popup)

  • anchor

    jQuery element which opens popup

  • event

    jQuery event or string (e. g. when use API method $.drop the parametr has value 'API')
